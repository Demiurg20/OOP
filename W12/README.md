# 3D Shape Modeling (Abstraction)

## About the Project

This project demonstrates abstraction in Python using 3D shapes.

The program creates different shapes and calculates:
- Surface area
- Volume

Shapes included:
- Sphere
- Cylinder
- Cube

---

## OOP Concepts Used

- Abstraction (abstract class Shape3D)
- Inheritance
- Polymorphism

---

## How it Works

The program:
- Generates 10 random shapes
- Calculates surface area and volume
- Displays results

---

## How to Run

Run:
 # python main.py

 
---

## Sample Output
Sphere (r=5)
Surface Area: 314.16
Volume: 523.60
