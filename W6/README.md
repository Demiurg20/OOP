# BMI Calculator (PyQt Application)

## 📌 About the Project

This project is a simple BMI Calculator application created using Python and PyQt5.

The application allows users to:
- Enter their weight (kg)
- Enter their height (meters)
- Calculate their Body Mass Index (BMI)
- See their BMI category (Underweight, Normal, Overweight, Obese)

The program follows the Department of Health and Human Services guidelines.

---

## 🧠 How BMI is Calculated

BMI formula:

BMI = weight / (height × height)

BMI Categories:
- Underweight: less than 18.5
- Normal: 18.5 – 24.9
- Overweight: 25 – 29.9
- Obese: 30 or more

---

## 🖥 Features

- Graphical User Interface (GUI) using PyQt5
- Input fields for weight and height
- Calculate button
- Displays BMI result
- Displays BMI status
- Menu bar with:
  - File → Clear
  - File → Exit
  - Help → How to Use

---

## ▶ How to Run the Program

1. Make sure Python 3 is installed.
2. Install PyQt5:
 ***pip install PyQt5***
3. Run the program:
 ***python bmi_calculator.py***
